AI Financial Statement Extractor

An AI Agentic workflow that extracts financial statements from annual reports and converts them into structured Excel datasets.

GitHub Repository Back to Portfolio

Problem Statement

Annual reports often exceed one hundred pages, while only a small subset contains the core financial statements required for analysis. Manually locating these statements and transcribing them into spreadsheets is repetitive, time-consuming, and susceptible to human error.

Solution

This project implements an AI Agentic workflow using CrewAI to automatically locate financial statements within annual report PDFs, extract their contents, calculate financial highlights, and generate a professionally formatted Excel workbook.

Workflow

The pipeline begins by identifying the pages containing the Statement of Financial Position, Statement of Profit or Loss, Statement of Cash Flows, and Statement of Changes in Equity. These pages are extracted into a separate PDF before AI agents convert each statement into structured JSON. A final stage computes financial highlights and exports everything into an Excel workbook.

Key Features

• Automatic financial statement page detection.
• Hybrid PDF text extraction.
• AI-powered table extraction.
• Financial highlights generation.
• Multi-sheet formatted Excel export.
• Modular CrewAI Flow architecture with deterministic Python utilities.

Results

The workflow was evaluated using annual reports from four publicly listed companies. Across all test cases, the system correctly identified the financial statement pages and accurately extracted the reported figures into structured outputs.

Challenges & Limitations

Financial reports vary significantly in formatting, page layouts, terminology, and table structures. The workflow is designed to handle a wide range of report styles, although extremely unconventional layouts may still require further prompt engineering or workflow refinement.
